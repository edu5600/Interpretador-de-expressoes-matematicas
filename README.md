# Interpretador-de-expressoes-matematicas
Trabalho Final de LFA:

------------------------------------------------------------------------------------

formato padrão da expressão:
( -3 + 2 ) * 5

operações suportadas:
adição +
subtração -
multiplicação *
divisão /
resto da divisão %
divisão inteira //

------------------------------------------------------------------------------------

Para o funcionamento normal do código é necessário rodar as seguintes intruções 
no terminal do python:

pip install PrettyPrintTree
pip install "cmd2==2.4.3"
